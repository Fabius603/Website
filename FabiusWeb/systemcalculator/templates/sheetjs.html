Strings
slice(a, b)

Schneidet den Bereich von a nach b aus einem String heraus. Wenn nur ein Wert gegeben ist, wird der String ab dieser Position genommen.

Eingabe: Ausgabe:
var text = "Postbote";
text = text.slice(3, 7) text = "tbot" | "Postbote"
text = text.slice(2) text = "stbote" | "Postbote"
text = text.slice(-7, -3) text = "tbot" | "Postbote"
text = text.slice(-2) text = "stbote" | "Postbote"
text = text.slice() text = "" | "Postbote"
substring(a, b)

Schneidet den Bereich von Position a nach Position b aus einem String heraus. Wenn nur ein Wert gegeben ist, wird der String ab dieser Position genommen.

Eingabe: Ausgabe:
var text = "Postbote";
text = text.substring(2, 4) text = "st" | "Postbote"
text = text.substring(2) text = "stbote" | "Postbote"
text = text.substring(-7, -3) text = "" | "Postbote"
text = text.substring(-2) text = "Postbote" | "Postbote"
text = text.substring() text = "Postbote" | "Postbote"
substr(a, b)

Startet an der Position a und gibt eine Anzahl an b Zeichen wieder. Ist a negativ, zählt er von hinten aber nicht rüchwärts.

Eingabe: Ausgabe:
var text = "Postbote";
text = text.substr(2, 4) text = "stbo" | "Postbote"
text = text.substr(2) text = "stbote" | "Postbote"
text = text.substr(-5, 4) text = "tbot" | "Postbote"
text = text.substr(-2) text = "te" | "Postbote"
text = text.substr() text = "Postbote" | "Postbote"
replace(a, b)

String wird nach Wert a durchsucht. Der erste gefundene Wert wird duch Wert b ersetzt.

Eingabe: Ausgabe:
var text = "Herr Bart hat einen Bart";
text = text.replace("Bart", "Müller") text = "Herr Müller hat einen Bart" | ein Wort ausgetauscht
text = text.replace("Wort", "Müller") text = "Herr Bart hat einen Bart" | kein Wort ausgetauscht
text = text.replace(/Bart/g, "Müller") text = "Herr Müller hat einen Müller" | jedes Wort wurde ausgetauscht
toUpperCase()

Jeder Buchstabe wird zu einem Großbuchstaben

Eingabe: Ausgabe:
var text = "Ich Bin Klein";
text = text.toUpperCase() text = "ICH BIN KLEIN" | Buchstaben sind alle Großbuchstaben
toLowerCase()

Jeder Buchstabe wird zu einem Kleinbuchstaben

Eingabe: Ausgabe:
var text = "Ich Bin Klein";
text = text.toLowerCase() text = "ich bin klein" | Buchstaben sind alle Kleinbuchstaben
concat()

Zwei oder mehr Strings werden miteinander Verbunden. Die Strings in den Klammern werden der Reihe nach aneinander gereiht.

Eingabe: Ausgabe:
var text1 = "String1";
var text2 = "String2";
var text3 = "String3";
text = text.concat(text1) text = "ich bin klein" | Buchstaben sind alle Kleinbuchstaben
